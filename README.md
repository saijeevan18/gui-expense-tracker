This project is a Java-based Expense Tracker application developed in two versions:

Console-based application
GUI-based application using Java Swing

It helps users manage daily expenses efficiently with features like tracking, filtering, and data persistence.

Features
Console Version
Add expenses (Amount, Category, Date)
View all expenses
Calculate total expenses
Filter by category
Find highest expense
File handling (data saved in file)

GUI Version (Java Swing)
User-friendly interface
Input fields for expense entry
Display using TextArea
Add and view expenses
Total expense calculation
